## 📖 项目概述

本软件是一款针对电力生产领域发电机组运行维护而开发的专业统计工具。旨在解决电厂在发电机漏氢量计算中存在的**计算过程繁琐、人工记录不便、历史数据溯源难**以及**报表生成效率低**等痛点。

通过建立标准化的数学计算模型和结构化数据库存储，实现漏氢数据的精准分析与闭环管理。支持Windows XP/7/10/11 操作系统。

## ✨ 主要功能模块说明

### 机组参数配置模块

* **功能描述**：系统支持多机组并行管理。专工可为不同机组设定关键物理参数，包括发电机容积、标准大气压及用于对比的基准氢温。
* **技术实现**：配置文件以机组名命名，独立存储于 config 目录下，确保各机组数据互不干扰。

### 漏氢计算与统计模块

* **计算算法**：利用补氢前后压差、温差及机组容积，换算至标准状态下的氢气体积 。通过计算两次补氢操作间的时间跨度，得出 24 小时平均漏氢速率。并且引入基准温度修正系数，消除环境温度波动对统计结果的影响。
* **自动化处理**：系统支持历史记录自动排序，每新增一条记录，程序会自动寻找上一条记录作为基准进行差值运算。

### 周期数据汇总分析

* **功能描述**：系统实时计算当前记录所属月份和年份的加权平均漏氢率 。
* **技术亮点**：采用时间加权算法，即根据每段统计周期的时长占总时长的比例进行加权，使年/月均值更能客观反映机组密封性能的变化趋势。

### 分级权限与安全控制

* **班组模式**：仅限查看、翻页及新增补氢记录，修改已有补氢记录，确保基础原始数据的完整性 。
* **专工模式**：需密码验证，密码为（chd123456），进入后可执行删除错误记录、修改机组核心参数、重命名机组、新增机组等关键操作。

### 报表自动化导出

* **功能描述**：一键生成指定月份的《漏氢率统计月报表》 。
* **格式要求**：生成的 Excel 报表对于漏氢率超过 16m³/d（预设阈值）的数据，报表会自动以红色字体标注。并自动计算并展示本月/本年的基准与补偿加权月均率。
